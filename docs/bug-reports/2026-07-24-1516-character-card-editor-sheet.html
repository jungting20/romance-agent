제약 및 미확인 항목

Playwright-test allowlist에 viewport resize와 request interception 도구가 없어 정확한 1280x800/375x800, 강제 지연 loading/saving, PATCH 500 failure/retry, PATCH 404 mutation-unavailable는 확인하지 못했다. 너비 결함은 실제 1280px desktop width에서 두 번 확정했으며 높이 800px 동작은 추론하지 않았다. 공식 validator는 기존 agent 설정 assertion 때문에 보고서를 직접 검사하지 못했다. 티켓은 생성 직후 자동 claim되어 fresh JSON의 ready 확인 조건을 유지하지 못했다. 제품 코드는 검사 후에도 수정하지 않았다.
